# 0x00 Why?

那么为什么同样是vim，neovim就比vim强大很多呢？

其实是因为neovim能安装很多基于lua的插件

# 0x01 安装步骤

MacOS用户可以使用Homebrew来安装

brew install nvim
1

没有安装过Homebrew的可以使用下方命令安装

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
1

Linux用户就不用教了()

# 0x02 lazy-vim的安装

首先我推荐先安装一个插件管理器lazy-nvim

我们可以使用下方的脚本来自动安装lazy-nvim，只要把它放在~/.config/nvim/init.lua里(需要先创建一个)，并用neovim打开此文件，使用:source %后，就会自动安装lazy-nvim，并且自动加载lazy-nvim，安装脚本只会在没有安装lazy-nvim时运行

local lazypath = vim.fn.stdpath("data") .. "/lazy/lazy.nvim"
if not (vim.uv or vim.loop).fs_stat(lazypath) then
  vim.fn.system({
    "git",
    "clone",
    "--filter=blob:none",
    "https://github.com/folke/lazy.nvim.git",
    "--branch=stable", -- latest stable release
    lazypath,
  })
end
vim.opt.rtp:prepend(lazypath)

local plugins = {}
local opts = {}

require("lazy").setup(plugins, opts)

# 0x03 插件的安装

以catppuccin插件为例，在init.lua中的plugins中添加

{ "catppuccin/nvim", name = "catppuccin", priority = 1000 }
1

然后再在文件中添加如下内容就安装好了

require("catppuccin").setup()
1

重新进入neovim，lazy-nvim会自动安装插件

因为catppuccin是一个colorscheme插件，所以还需要在init.lua中添加

vim.cmd.colorscheme "catppuccin"
1

此时再进入neovim，就可以了
